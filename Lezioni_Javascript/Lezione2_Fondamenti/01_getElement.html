Get Element By Id

Con Javascript posso manipolare tutta la pagina HTML. La cosa più semplice che possiamo fare è scrivere all'interno di un div o altro elemento della pagina. Succivamente impareremo a modificare gli attributi dei singoli elementi

Per adesso utilizzerò l'attributo id per singolo elemento. Questo id mi servirà nello script per riuscire a "raccogliere" l'elemento e modificare il suo contenuto
