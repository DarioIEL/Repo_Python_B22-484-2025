# Esercizi con Array

### Es 1
Costruisci uno scontrino della spesa. Avrai a disposizione 2 array: 1 con la lista delle cose acquistate; l'altro con i relativi prezzi. (10 cose) 
Al fondo calcola il totale dello scontrino.
Mettilo in una pagina Html alla quale fornire delle regole di stile per poter vedere meglio lo scontrino. 

### Es 2
Crea un array di parole e stampalo nella console. Stampa l'array nella pagina. Ogni parola dovrà essere inserita in un tag <li>

### Es 3
Crea un array di numeri ( 12 numeri ). 
Crea una pagina html con 3 div.
All'interno di ogni div stampa la lista di numeri in utilizzando un for diverso. 
Nell'ultimo div sistema la lista di numeri in ordine crescente.
