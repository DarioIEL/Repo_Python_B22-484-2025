# Pagine Web "Città del mondo"
Elabora il seguente sito con le seguenti caratteristiche:
 - una pagina Home nella quale presenti le tue città preferite in una lista 
 - per ogni elemento della lista imposta un link che riconduca alla relativa pagina della città 

## Pagina singola città
Nella pagina della singola città deve essere presente una foto con accanto un testo di copertura. Cerca più o meno di rispettare una proporzione di 60-40 (60% immagine, 40% testo)

Sotto ogni immagine un link "Indietro" che ti riporterà alla pagina iniziale
