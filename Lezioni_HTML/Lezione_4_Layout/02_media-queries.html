Design Responsive

Il design responsive è un approccio alla progettazione che mira a rendere la fruizione e l'utilizzo della pagina ottimali. Il nostro sito web dovrà essere ottimale a prescindere dal dispositivo (smartphone, tablet, desktop)

Fare Design responsive vuol dire progettare un sito web pensandolo "Mobile First" prima per lo smartphone

Le Media Queries cono funzionalità del nostro css che consentono di applicare degli stili specifici in base alle caratteristiche del dispositivo sul quale visualizziamo la pagina. Questi stili si applicano utilizzano dei breakpoint
