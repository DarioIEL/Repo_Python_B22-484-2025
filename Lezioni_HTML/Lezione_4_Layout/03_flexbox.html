Flexbox, concetti chiave

Flexbox va benissimo per: navbar, gruppi di pulsanti, card, elementi perfettamente centrati. Se ho da sviluppare un layout complesso mi serve css grid

Flexbox evita di utilizzare regole un po' macchinose come float oppure inline-block

Il container flex: display:flex e flex-direction

Il primo passo è trasformare un elemento in flex container usando la regola display: flex. Poi scegli la direzione principale andando quindi ad impostare il main axis. row (default, orizzontale) oppure posso utilizzare column (verticale)
